## Commands & Mappings by Moi
> N.B. my leader key is **","**

<br>

1. **Change colorscheme**
```vim
:lua ChangeBG("<color>")
:lua ChangeBG() " default = rose-pine
```

2. **File & Git Related**
```vim
,pv     " open path view
,ff     " file-finder
,gf     " file-finder across git repos
,gs     " do git stuff
,ps     " file-finding in project
```

3. **UndoTree**
```vim
,u  " toggle undotree
```

## Plugin Commands

1. **Markdown-Preview**
```vim
" original
:MarkdownPreview
:MarkdownPreviewStop

" remaps
,md
,mds
```


2. **Packer**
```vim
:so
:PackerSync
```


3. **Nvim-Tree**
```vim
" original
:NvimTreeOpen
:NvimTreeClose
:NvimTreeToggle

" remaps
,tt
```


4. **ToggleTerm**
```vim
" original
:ToggleTerm size=40 dir=~/<directory> direction=<horizontal/vertical> name=<name>

" remaps

```
